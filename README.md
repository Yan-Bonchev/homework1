# Мини библиотечна система (C++)

Проектът представлява малка библиотечна система, реализирана на C++, която демонстрира основни обектно-ориентирани принципи: капсулация, конструктори и деструктори, статични членове, валидация на данни и прилагане на правилото Rule of 3/5

## Компилация

Използвайте следната команда:
`g++ -std=gnu++17 -Wall -Wextra -O2 -o library main.cpp Library.cpp Book.cpp`

Класове:
- `Author` – съдържа име и година на раждане; включва валидация и метод to_string().
- `Book` – съдържа заглавие, автор, година, цена и ISBN; включва статичен брояч на книгите и Rule of 5
- `Member` – представя читател с име, ID и година на присъединяване
- `Loan` – заем на книга с начална и крайна дата; включва markReturned() и проверка за просрочване
- `Library` – колекция от книги, членове и заеми; позволява добавяне, заемане, връщане и търсене

Изход:
```
Library{books=2, members=1, activeLoans=0}
Loan created.
Available ISBN-001? false
Available ISBN-001? true
Book{title="Pod igoto", author="Ivan Vazov", year=1894, price=25.5, isbn="ISBN-001"}
Book{title="Nema zemya", author="Ivan Vazov", year=1900, price=18.9, isbn="ISBN-002"}
```
